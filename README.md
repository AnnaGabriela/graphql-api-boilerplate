# GraphQL API

``` bash
# clone the project
$ git clone https://github.com/AnnaGabriela/graphql-api-boilerplate

# install the dependencies
$ npm install

# run
$ npm server.js
``` 

**Obs:** *The GraphiQL is enabled by default and it's available at 'localhost:PORT/graphiql'*
